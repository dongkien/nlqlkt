Python Lab 0 — Cài Python + pandas + Jupyter

Nền tảng trước khi vào 15 lab buổi

Note🎯 Mục tiêu Lab 0

Sau Lab 0, bạn có thể:

  • Cài Python + Jupyter trên Windows/macOS/Linux (hoàn toàn miễn phí)
  • Cài các package cho phân tích kinh tế: pandas, numpy, matplotlib, statsmodels
  • Tải, đọc, ghi dataset (.csv, .xlsx, .dta)
  • Khám phá nhanh dữ liệu bằng .info(), .describe(), .value_counts()
  • Viết script Python đầu tiên có comment tiếng Việt
  • Báo cáo bằng Jupyter Notebook hoặc Quarto

1. Cài Python + môi trường

Phương án A — Anaconda (khuyên dùng cho người mới)

Anaconda = Python + 100+ package khoa học + Spyder IDE + Jupyter, tất cả trong 1 lần cài.

OS Đường dẫn
Windows https://www.anaconda.com/download → tải .exe
macOS https://www.anaconda.com/download → tải .pkg cho chip Intel hoặc Apple Silicon
Linux/WSL wget https://repo.anaconda.com/archive/Anaconda3-latest-Linux-x86_64.sh && bash Anaconda3-latest-Linux-x86_64.sh

Sau khi cài, mở Anaconda Navigator — bạn có thể chọn: - Jupyter Lab — môi trường tương tác (khuyên dùng) - Spyder — IDE giống RStudio - VS Code — IDE đa năng

Phương án B — Python.org + pip (gọn hơn)

Nếu bạn muốn nhẹ:

  1. Cài Python: https://www.python.org/downloads/
  2. Cài Jupyter + packages trong Terminal:
pip install jupyterlab pandas numpy matplotlib seaborn statsmodels scikit-learn openpyxl pyreadstat linearmodels
  1. Chạy Jupyter Lab:
jupyter lab

Phương án C — Google Colab (không cài đặt)

Nếu không muốn cài gì:

  • Vào https://colab.research.google.com — đăng nhập Google
  • Tạo notebook mới — Python chạy trên cloud
  • Free unlimited cho tasks nhỏ
  • Có sẵn pandas, numpy, matplotlib, statsmodels

2. Cài thêm các package cho kinh tế

Trong Jupyter Lab → tạo notebook mới → trong cell đầu:

# Cài package (chỉ chạy 1 lần — chỉ dùng trong Jupyter)
%pip install pandas numpy matplotlib seaborn statsmodels scikit-learn
%pip install openpyxl       # đọc Excel .xlsx
%pip install pyreadstat     # đọc Stata .dta
%pip install linearmodels   # panel data, IV
%pip install wbdata         # World Bank API

%pip install ... là Jupyter magic command — chạy pip install trong môi trường đúng. Chỉ hoạt động trong notebook (.ipynb), không hoạt động trong file .py thuần.

Nếu chạy từ Terminal (file .py): dùng pip install trực tiếp trong bash (không phải Python):

pip install pandas numpy matplotlib seaborn statsmodels scikit-learn openpyxl pyreadstat linearmodels wbdata

Đợi vài phút. Sau khi cài xong, restart kernel (Kernel > Restart).

3. Script Python đầu tiên

Tạo notebook mới → đặt tên lab00_<ten>.ipynb.

# Lab 0 — Buổi học đầu tiên với Python
# Họ tên: <điền tên>
# MSSV: <điền MSSV>

# Import các thư viện
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
import seaborn as sns

# Dataset có sẵn của seaborn: mpg (giống mtcars của R)
mpg = sns.load_dataset("mpg")

# In thông báo
print("Python đang chạy. Bạn đã mở dataset mpg")

# Xem 5 dòng đầu
mpg.head()

4. Khám phá dataset — bốn lệnh cốt lõi

.info() — mô tả cấu trúc

mpg.info()

Output: - Số dòng + số cột - Kiểu dữ liệu từng cột (int64, float64, object…) - Số missing mỗi cột

.describe() — thống kê mô tả

mpg.describe()                          # tất cả cột số
mpg[["mpg", "horsepower", "weight"]].describe()  # chỉ 3 cột

Cho: count, mean, std, min, 25%, 50%, 75%, max.

.value_counts() — bảng tần suất

mpg["origin"].value_counts()            # đếm theo xuất xứ
mpg["cylinders"].value_counts()         # đếm theo xi-lanh
pd.crosstab(mpg["origin"], mpg["cylinders"])  # bảng chéo

Xem dòng dữ liệu

mpg.head(10)                                  # 10 dòng đầu
mpg[mpg["mpg"] > 30]                          # mpg > 30
mpg[mpg["origin"] == "japan"][["name", "mpg"]]  # tên + mpg xe Nhật
Tip📝 Bài tập nhỏ ngay tại Lab 0

Mở dataset mpg, trả lời:

  1. Có bao nhiêu xe trong dataset?
  2. mpg (mile per gallon) trung bình là bao nhiêu?
  3. Có bao nhiêu xe theo origin (usa/europe/japan)?
  4. Xe nào có mpg cao nhất?
  5. So sánh trung bình mpg giữa xe Mỹ và xe Nhật?

5. Tạo cột + lọc + sort

# Tạo cột mới
mpg["gpm"] = 1 / mpg["mpg"]                   # gallon per mile
mpg["is_efficient"] = mpg["mpg"] > 25         # xe tiết kiệm

# Lọc + select cột
mpg_filtered = (
    mpg[mpg["cylinders"] != 4]                # bỏ xe 4 xi-lanh
    [["mpg", "horsepower", "gpm", "is_efficient"]]  # giữ 4 cột
    .sort_values("mpg", ascending=False)      # sắp theo mpg giảm
)

mpg_filtered.head(10)

groupby — bysort của Stata

# Trung bình mpg theo origin
mpg.groupby("origin")["mpg"].mean()

# Nhiều stat cùng lúc
mpg.groupby("origin")["mpg"].agg(["mean", "std", "min", "max"])

# Theo 2 nhóm
mpg.groupby(["origin", "cylinders"])["mpg"].mean()

6. Vẽ biểu đồ — matplotlib + seaborn

Scatter plot

import matplotlib.pyplot as plt
import seaborn as sns

# Cấu hình style
sns.set_theme(style="whitegrid")

fig, ax = plt.subplots(figsize=(10, 6))
sns.scatterplot(data=mpg, x="mpg", y="horsepower", hue="origin", ax=ax)
ax.set_title("Công suất vs Tiết kiệm xăng")
ax.set_xlabel("Mile per gallon")
ax.set_ylabel("Horsepower")
plt.tight_layout()
plt.savefig("scatter.png", dpi=150)
plt.show()

Bar chart theo nhóm

fig, ax = plt.subplots(figsize=(8, 5))
mpg["origin"].value_counts().plot(
    kind="bar", ax=ax, color="#722f37"
)
ax.set_title("Số xe theo xuất xứ")
ax.set_ylabel("Số xe")
plt.tight_layout()
plt.savefig("bar.png", dpi=150)
plt.show()

Histogram

fig, ax = plt.subplots(figsize=(8, 5))
sns.histplot(data=mpg, x="mpg", bins=20, color="#c19a4d", ax=ax)
ax.set_title("Phân bố MPG")
plt.tight_layout()
plt.savefig("hist.png", dpi=150)
plt.show()

10. Mẹo Python hiệu quả

  1. Luôn import packages đầu file — gọn, dễ debug
  2. Dùng pd.options.display.max_columns = None để xem hết cột
  3. Comment tiếng Việt được — Python hỗ trợ UTF-8 đầy đủ
  4. Đặt biến df cho DataFrame chính — phổ biến trong cộng đồng
  5. Save dataset thường xuyên sau khi clean
  6. Method chaining: dùng dấu ngoặc () để xuống dòng đẹp
  7. Help: gõ pd.read_csv? hoặc help(pd.read_csv) để xem docs

11. Tích hợp với Stata MP (tuỳ chọn)

Nếu bạn có Stata, Python có thể gọi Stata qua PyStata:

import sys
sys.path.insert(0, '/usr/local/stata19/utilities')
import pystata
pystata.config.init('mp', splash=False)

# Truyền DataFrame → Stata
pystata.stata.pdataframe_to_data(mpg, force=True)

# Chạy lệnh Stata
pystata.stata.run("""
    regress mpg horsepower weight
""")

Đặc điểm: tận dụng hệ sinh thái Python (visualization, ML) + engine Stata (lệnh kinh tế lượng quen).
