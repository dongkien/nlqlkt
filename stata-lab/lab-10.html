Lab 10 — Mundell-Fleming bằng dữ liệu lịch sử

Tương ứng Buổi 10: Hong Kong LERS, Reaganomics, ERM 1992

Note🎯 Liên kết với Buổi 10

Phân tích 3 case lịch sử bằng dữ liệu macro: Hong Kong LERS (peg cứng), Reaganomics 1980s (twin deficits), ERM 1992 (Black Wednesday).

Khái niệm Stata học: panel data setup, time-varying analysis, conditional statements

1. Tạo dataset 3 nền kinh tế

* Lab 10 — Mundell-Fleming historical cases
clear all
set more off
log using "lab10_<ten>.log", replace text

clear
set obs 30                      // 3 nước × 10 năm
generate id = ceil(_n/10)       // 1=HK, 2=Mỹ Reagan, 3=Anh ERM
generate year = mod(_n-1, 10)

* === HK 2015-2024 ===
replace year = 2015 + year if id == 1
* === Mỹ 1980-1989 ===
replace year = 1980 + year if id == 2
* === Anh 1985-1994 ===
replace year = 1985 + year if id == 3

generate country = ""
replace country = "Hong Kong" if id == 1
replace country = "USA"       if id == 2
replace country = "UK"        if id == 3

* Tỷ giá thực
generate REER = .
* Hong Kong: ổn định nhờ peg
replace REER = 100 + 2*(year - 2015) if id == 1

* Mỹ thời Reagan: lên giá mạnh
replace REER = 100             if id == 2 & year == 1980
replace REER = 110             if id == 2 & year == 1981
replace REER = 125             if id == 2 & year == 1982
replace REER = 140             if id == 2 & year == 1983
replace REER = 155             if id == 2 & year == 1984
replace REER = 150             if id == 2 & year == 1985
replace REER = 130             if id == 2 & year == 1986
replace REER = 115             if id == 2 & year == 1987
replace REER = 105             if id == 2 & year == 1988
replace REER = 102             if id == 2 & year == 1989

* Anh ERM: cao trước 1992, sụp 1992
replace REER = 100             if id == 3 & year == 1985
replace REER = 102             if id == 3 & year == 1986
replace REER = 105             if id == 3 & year == 1987
replace REER = 108             if id == 3 & year == 1988
replace REER = 110             if id == 3 & year == 1989
replace REER = 113             if id == 3 & year == 1990
replace REER = 115             if id == 3 & year == 1991
replace REER = 85              if id == 3 & year == 1992
replace REER = 80              if id == 3 & year == 1993
replace REER = 78              if id == 3 & year == 1994

* CA / GDP (%)
generate CA = .
replace CA = 5  if id == 1                                  // HK luôn thặng dư
replace CA = -1 if id == 2 & year == 1980
replace CA = -1 if id == 2 & year == 1981
replace CA = -2 if id == 2 & year == 1982
replace CA = -3 if id == 2 & year == 1983
replace CA = -3 if id == 2 & year == 1984
replace CA = -3 if id == 2 & year == 1985
replace CA = -3 if id == 2 & year == 1986
replace CA = -3 if id == 2 & year == 1987
replace CA = -2 if id == 2 & year == 1988
replace CA = -2 if id == 2 & year == 1989
replace CA = -2 if id == 3                                  // Anh
replace CA = -4 if id == 3 & year >= 1989 & year <= 1991
replace CA = -1 if id == 3 & year >= 1993

list, sepby(country)

2. Phân tích case-by-case

* === Hong Kong: peg + tỷ giá ổn định ===
display "=== Hong Kong LERS ==="
summarize REER if country == "Hong Kong"
display "REER std dev rất thấp do peg cứng"

* === Mỹ Reaganomics ===
display "=== Reaganomics ==="
summarize REER if country == "USA"
display "REER lên giá ~55% giữa 1980-1985 → twin deficits"

* === ERM 1992 ===
display "=== ERM 1992 — Anh ==="
summarize REER if country == "UK"
display "REER giảm mạnh 1992 → Black Wednesday"

Câu hỏi 1: REER HK ổn định nhất do cơ chế nào? (Gợi ý: peg cứng + dự trữ 3x M3)

Câu hỏi 2: Mỹ Reagan REER tăng 55% trong 5 năm — phản ánh CSTK + CSTT phối hợp như thế nào?

Câu hỏi 3: Anh ERM REER giảm 30% năm 1992 trong 1 ngày — Soros bán khống GBP với quy mô bao nhiêu?
