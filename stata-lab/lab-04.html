Lab 4 — Mô phỏng thuế Pigou

Tương ứng Buổi 4: Cơ chế điều tiết ngoại tác và tính toán DWL

Note🎯 Liên kết với Buổi 4

Bài này áp dụng khái niệm thuế Pigou từ Buổi 4 vào một ngành có ngoại tác âm điển hình tại Việt Nam: xi măng (phát thải CO₂).

Sau Lab 4, bạn có thể:

  • Tạo dataset cung-cầu trong Stata
  • Tìm điểm cân bằng thị trường (giao điểm S, D)
  • So sánh MPC (chi phí tư nhân biên) và MSC (chi phí xã hội biên)
  • Áp thuế Pigou = MEC và tìm cân bằng mới
  • Tính DWL (deadweight loss) trước và sau thuế

Khái niệm Stata học: set obs, generate, mô phỏng đại số, twoway nhiều layer

1. Bài toán

Ngành xi măng VN có:

  • Đường cầu: \(Q_D = 100 - P\)
  • Đường cung tư nhân (MPC): \(P = 20 + 0.5Q\)
  • Chi phí ngoại tác biên (MEC) từ phát thải CO₂: \(\text{MEC} = 0.5Q\)
  • Chi phí xã hội biên (MSC) = MPC + MEC = \(20 + Q\)

Câu hỏi:

  1. Cân bằng thị trường tự do (không thuế) ở đâu?
  2. Cân bằng xã hội tối ưu (có MSC) ở đâu?
  3. Thuế Pigou bằng bao nhiêu?
  4. DWL trước thuế là bao nhiêu?

2. Tạo dataset cung-cầu

* Lab 4 — Mô phỏng thuế Pigou cho ngành xi măng
* Họ tên: <điền>, MSSV: <điền>

clear all
set more off
log using "lab04_<ten>.log", replace text

* === Tạo lưới Q từ 0 đến 100 ===
clear
set obs 101
generate Q = _n - 1

* === Tính các đường ===
generate P_D = 100 - Q                    // Cầu: P = 100 - Q
generate P_MPC = 20 + 0.5 * Q             // Cung tư nhân (Marginal Private Cost)
generate P_MEC = 0.5 * Q                  // Chi phí ngoại tác biên
generate P_MSC = P_MPC + P_MEC            // Chi phí xã hội biên = MPC + MEC

label variable P_D "Cầu D"
label variable P_MPC "Cung tư nhân (MPC)"
label variable P_MEC "Chi phí ngoại tác (MEC)"
label variable P_MSC "Chi phí xã hội (MSC)"

list Q P_D P_MPC P_MSC in 1/10

3. Tìm cân bằng thị trường tự do

Cân bằng khi \(P_D = P_{MPC}\):

\[100 - Q = 20 + 0.5Q \Rightarrow Q^* = 53.33,\, P^* = 46.67\]

Kiểm tra bằng Stata:

* === Cân bằng thị trường tự do (không có thuế) ===
generate dif_tu_do = abs(P_D - P_MPC)
sort dif_tu_do
display "Q* tự do  ≈ " Q[1]
display "P* tự do  ≈ " P_D[1]

* Lưu giá trị
scalar Q_tu_do = Q[1]
scalar P_tu_do = P_D[1]
sort Q

Câu hỏi 1: Cân bằng thị trường tự do ở Q ≈ ? và P ≈ ?

4. Tìm cân bằng xã hội tối ưu

Khi tính cả ngoại tác, cân bằng đúng là khi \(P_D = P_{MSC}\):

\[100 - Q = 20 + Q \Rightarrow Q^{**} = 40,\, P^{**} = 60\]

* === Cân bằng xã hội tối ưu (có ngoại tác) ===
generate dif_xa_hoi = abs(P_D - P_MSC)
sort dif_xa_hoi
display "Q** xã hội ≈ " Q[1]
display "P** xã hội ≈ " P_D[1]

scalar Q_xh = Q[1]
scalar P_xh = P_D[1]
sort Q

Câu hỏi 2: Cân bằng xã hội ở Q ≈ ? và P ≈ ? So với tự do: sản lượng giảm bao nhiêu? Giá tăng bao nhiêu?

5. Thuế Pigou — bằng MEC tại điểm tối ưu

* === Thuế Pigou tại Q_xh ===
* Pigou tax = MEC khi Q = Q**
sort Q
scalar pigou_tax = 0.5 * Q_xh
display "Thuế Pigou = MEC tại Q** = " pigou_tax

* Kiểm tra: cung sau thuế = MPC + thuế = MSC
generate P_MPC_co_thue = P_MPC + pigou_tax

* Cân bằng mới: D = MPC + thuế
generate dif_sau_thue = abs(P_D - P_MPC_co_thue)
sort dif_sau_thue
display "Q sau thuế ≈ " Q[1]
display "P sau thuế ≈ " P_D[1]
sort Q

Câu hỏi 3: Thuế Pigou bằng bao nhiêu? Sau khi áp thuế, cân bằng có trùng với cân bằng xã hội tối ưu (Q** = 40, P** = 60) không?

6. Tính DWL (Deadweight Loss)

DWL = diện tích tam giác mất phúc lợi do thị trường tự do sản xuất quá nhiều.

Hình tam giác có:

  • Đáy: \(\Delta Q = Q^* - Q^{**} = 53.33 - 40 = 13.33\)
  • Chiều cao: chênh lệch MSC – MPC tại \(Q^*\) = \(0.5 \times 53.33 = 26.67\)

\[\text{DWL} = \frac{1}{2} \times 13.33 \times 26.67 \approx 177.8\]

Tính bằng Stata:

* === Tính DWL ===
scalar delta_Q = Q_tu_do - Q_xh
scalar MEC_at_Qfree = 0.5 * Q_tu_do
scalar DWL = 0.5 * delta_Q * MEC_at_Qfree
display "Đáy ΔQ = " delta_Q
display "Cao MEC = " MEC_at_Qfree
display "DWL = " DWL

Câu hỏi 4: DWL bằng bao nhiêu? Nếu tăng được thuế Pigou đúng, toàn bộ DWL có được loại bỏ không?
