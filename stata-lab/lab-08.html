Lab 8 — Đường Phillips Việt Nam 2010-2024

Tương ứng Buổi 8: Ước lượng OLS, so sánh ngắn hạn từng giai đoạn

Note🎯 Liên kết với Buổi 8

Bài này áp dụng đường Phillips từ Buổi 8 vào dữ liệu Việt Nam 2010-2024. Bạn sẽ kiểm tra: Phillips có còn tồn tại ở VN không? và Liệu kỳ vọng có làm dịch chuyển Phillips ngắn hạn?

Sau Lab 8, bạn có thể:

  • Tạo dataset lạm phát + thất nghiệp VN
  • Vẽ scatter Phillips
  • Ước lượng OLS độ dốc Phillips
  • So sánh Phillips trước Covid (2010-2019) vs sau (2020-2024)
  • Liên hệ với mô hình kỳ vọng Friedman-Phelps

Khái niệm Stata học: scatter + fitted line, regress by group, if condition, twoway combine

1. Dataset CPI và thất nghiệp Việt Nam

* Lab 8 — Đường Phillips Việt Nam 2010-2024
* Họ tên: <điền>, MSSV: <điền>

clear all
set more off
log using "lab08_<ten>.log", replace text

* === Dữ liệu lạm phát + thất nghiệp VN ===
* Nguồn: TCTK (CPI YoY), ước lượng tỷ lệ thất nghiệp khu vực thành thị
clear
set obs 15

generate year = 2010 + _n - 1

* Lạm phát (CPI YoY, %) - ước lượng
generate inflation = .
replace inflation = 9.2  if year == 2010
replace inflation = 18.7 if year == 2011  // sốc lạm phát
replace inflation = 9.1  if year == 2012
replace inflation = 6.6  if year == 2013
replace inflation = 4.1  if year == 2014
replace inflation = 0.6  if year == 2015
replace inflation = 2.7  if year == 2016
replace inflation = 3.5  if year == 2017
replace inflation = 3.5  if year == 2018
replace inflation = 2.8  if year == 2019
replace inflation = 3.2  if year == 2020  // Covid
replace inflation = 1.8  if year == 2021
replace inflation = 3.2  if year == 2022
replace inflation = 3.3  if year == 2023
replace inflation = 3.6  if year == 2024

* Thất nghiệp khu vực thành thị (%) - ước lượng
generate unemployment = .
replace unemployment = 4.4  if year == 2010
replace unemployment = 3.6  if year == 2011
replace unemployment = 3.2  if year == 2012
replace unemployment = 3.6  if year == 2013
replace unemployment = 3.4  if year == 2014
replace unemployment = 3.4  if year == 2015
replace unemployment = 3.2  if year == 2016
replace unemployment = 3.2  if year == 2017
replace unemployment = 3.1  if year == 2018
replace unemployment = 3.0  if year == 2019
replace unemployment = 3.9  if year == 2020  // Covid
replace unemployment = 4.4  if year == 2021
replace unemployment = 3.0  if year == 2022

Dữ liệu mô phỏng xấp xỉ thực tế. Nguồn chính thức: - TCTK (gso.gov.vn) — CPI và Bảng KSDN - Bộ LĐ-TB-XH — Báo cáo thị trường LĐ - WB Vietnam Development Report

2. Vẽ scatter Phillips toàn giai đoạn

* === Phillips scatter cơ bản ===
twoway (scatter inflation unemployment, mlabel(year) mlabsize(small) ///
        msize(medium) mcolor("114 47 55")) ///
       (lfit inflation unemployment, lcolor("0 88 163") lwidth(medium)), ///
       title("Đường Phillips Việt Nam, 2010-2024") ///
       xtitle("Tỷ lệ thất nghiệp (%)") ///
       ytitle("Lạm phát (%)") ///
       legend(off) ///
       graphregion(color(white))

graph export "phillips_full.png", replace width(1400)

Câu hỏi 1: Đường fitted có dốc xuống (Phillips tồn tại) hay dốc lên (Phillips bị phá vỡ)? 2011 là outlier hay đại diện?

3. Ước lượng độ dốc Phillips toàn giai đoạn

* === OLS toàn giai đoạn ===
display "=== Phillips 2010-2024 toàn giai đoạn ==="
regress inflation unemployment, robust

Câu hỏi 2: Hệ số unemployment ước lượng được bao nhiêu? Có ý nghĩa thống kê 5% không? Theo lý thuyết Phillips, kỳ vọng dấu âm — có đúng không?

4. So sánh hai giai đoạn

Theo Friedman-Phelps, kỳ vọng làm dịch Phillips. Liệu Phillips 2010-2019 có khác Phillips 2020-2024?

* === Tạo biến giai đoạn ===
generate giai_doan = ""
replace giai_doan = "2010-2019" if year <= 2019
replace giai_doan = "2020-2024" if year >= 2020

* === Regress theo 2 giai đoạn ===
display "=== Phillips 2010-2019 (trước Covid) ==="
regress inflation unemployment if year <= 2019, robust

display "=== Phillips 2020-2024 (sau Covid) ==="
regress inflation unemployment if year >= 2020, robust

Câu hỏi 3: Độ dốc Phillips hai giai đoạn có khác nhau không? Giai đoạn nào dốc hơn? Ý nghĩa?

5. Vẽ scatter có phân biệt giai đoạn

* === Scatter có màu theo giai đoạn ===
twoway (scatter inflation unemployment if year <= 2019, mcolor("114 47 55") ///
        mlabel(year) mlabsize(small) msize(medium)) ///
       (scatter inflation unemployment if year >= 2020, mcolor("0 88 163") ///
        mlabel(year) mlabsize(small) msize(medium)) ///
       (lfit inflation unemployment if year <= 2019, lcolor("114 47 55") ///
        lwidth(medium) lpattern(dash)) ///
       (lfit inflation unemployment if year >= 2020, lcolor("0 88 163") ///
        lwidth(medium) lpattern(dash)), ///
       title("Phillips VN: 2010-2019 vs 2020-2024") ///
       xtitle("Tỷ lệ thất nghiệp (%)") ytitle("Lạm phát (%)") ///
       legend(label(1 "2010-2019") label(2 "2020-2024") ///
              order(1 2) position(2) ring(0) cols(1)) ///
       graphregion(color(white))

graph export "phillips_2periods.png", replace width(1400)

6. Phillips kỳ vọng — đơn giản

Theo Friedman-Phelps: \[\pi_t = \pi^e_t - \beta(U_t - U^N)\]

Một cách đơn giản: kỳ vọng = lạm phát năm trước.

* === Lagged inflation làm proxy cho expectations ===
sort year
generate inflation_lag = inflation[_n-1]
generate inflation_expected = inflation_lag    // Naive expectations

* === Phillips expectation-augmented ===
regress inflation unemployment inflation_lag if year >= 2011, robust

Câu hỏi 4: Khi thêm inflation_lag (proxy cho kỳ vọng), độ dốc của unemployment có thay đổi không? Có gần với NAIRU lý thuyết hơn không?

7. Tính NAIRU ước lượng

Theo công thức: \[U^N = U - \frac{\pi - \pi^e}{\beta}\]

Với β = độ dốc Phillips, \(\pi^e\) = inflation_lag:

* === Ước lượng NAIRU ===
* Lấy beta từ hồi quy vừa rồi
local beta = -_b[unemployment]
display "Beta = " `beta'

generate NAIRU = unemployment - (inflation - inflation_lag) / `beta'

* Trung bình NAIRU
summarize NAIRU, detail
display "NAIRU trung bình = " r(mean) "%"

Câu hỏi 5 (suy nghĩ + viết, 400 từ):

NAIRU của Việt Nam trung bình bao nhiêu? Có hợp lý không? (gợi ý: tỷ lệ thất nghiệp chính thức VN luôn rất thấp ~2-4%, có thể NAIRU cũng vậy.)

Theo bạn, Phillips có còn áp dụng được ở Việt Nam sau giai đoạn 2010 không? Lập luận của bạn dựa trên:

  • Cú sốc cung 2022 (Nga-Ukraine) làm lạm phát tăng nhưng thất nghiệp không giảm — đây có phải bằng chứng Phillips bị phá vỡ?
  • Năm 2011 lạm phát cực cao (18.7%) nhưng thất nghiệp vẫn thấp — vì sao?
  • Số liệu thất nghiệp VN có chính xác không? (gợi ý: kinh tế phi chính thức, thiếu việc làm)
