Stata Lab

Tự học Stata qua 15 bài thực hành — tương ứng 15 buổi của môn

Stata Lab là gì?

Stata Lab là chuỗi 15 bài thực hành tự nguyện đi kèm môn . Mỗi lab tương ứng với một buổi giảng, lấy dữ liệu thực tế Việt Nam và quốc tế để vận dụng khái niệm vừa học.

Mục tiêu:

  • Nắm vững Stata ở mức cử nhân năm 3-4 — đủ để làm khoá luận tốt nghiệp, làm việc tại các viện nghiên cứu, hoặc tiếp tục bậc thạc sĩ.
  • Cảm nhận lý thuyết bằng dữ liệu thật — thấy Cobb-Douglas, đường Phillips, Bộ ba bất khả thi bằng số chứ không chỉ bằng chữ.
  • Tự xây dựng bộ tài sản kỹ năng (do-file, ghi chú) có thể dùng dài hạn.
Tip🔄 Cũng có: R Lab + Python Lab song song

3 phiên bản cùng nội dung — sinh viên chọn 1 trong 3 hoặc làm nhiều hơn để bonus cao hơn (1 lab: +5%, 2 lab: +7%, 3 lab: +9%).

  • → R Lab — tidyverse + ggplot2 (free)
  • → Python Lab — pandas + matplotlib (free, mạnh nhất cho data science)

🎁 Cơ chế bonus

Nộp Lab (Stata/R/Python) có thể cộng tới 0,5–0,9 điểm vào điểm tổng kết (1 lab → 5%, 2 lab → 7%, 3 lab → 9% bonus). Xem Đánh giá để biết cách quy đổi.

Cách nộp Stata Lab

Mỗi lab có deliverables rõ ràng:

  1. File .do chứa code đầy đủ, có comment tiếng Việt giải thích từng đoạn
  2. File .log chứa output (kết quả chạy)
  3. File .pdf ngắn (1-2 trang) trả lời các câu hỏi suy nghĩ trong lab

Tiêu đề email: LAB[số]-MSSV-HọTên → gửi giảng viên trước hạn nộp công bố đầu kỳ.

Rubric chấm (đơn giản)

Mỗi lab chấm theo 4 mức:

Mức Mô tả Điểm bonus
Đạt xuất sắc Code chạy đúng, có comment rõ, trả lời câu hỏi sâu, có liên hệ thực tiễn 0.5 điểm/lab
Đạt tốt Code chạy đúng, trả lời câu hỏi đầy đủ 0.3 điểm/lab
Đạt cơ bản Code chạy, có nỗ lực trả lời câu hỏi 0.1 điểm/lab
Không đạt Code không chạy hoặc không nộp 0 điểm

Tổng tối đa: ~0,5 điểm vào điểm tổng kết (giảng viên có thể điều chỉnh tỷ lệ).

Important
  • Làm để hiểu, không phải để cộng điểm. Mục tiêu thật sự là kỹ năng dài hạn.
  • Bắt đầu từ Lab 0 — đừng nhảy. Stata có cú pháp riêng, cần làm quen.
  • Không sao chép code mà không hiểu. Mỗi dòng code phải biết nó làm gì.
  • Sử dụng AI có trách nhiệm: ChatGPT/Claude có thể giải thích lệnh, nhưng không thay bạn nghĩ. Khai báo nếu AI hỗ trợ.
  • Lưu giữ lab của bạn — đây là bộ tài sản dài hạn.
