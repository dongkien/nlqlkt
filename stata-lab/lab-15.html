Lab 15 — Xuất bảng + biểu đồ cho thuyết trình

Tương ứng Buổi 15: outreg2, coefplot, esttab — chuyên nghiệp hoá kết quả

Note🎯 Liên kết với Buổi 15

Lab cuối cùng: xuất kết quả Stata ra Word/LaTeX/Excel đẹp dùng cho thuyết trình. Đây là kỹ năng bắt buộc khi viết khoá luận hoặc paper học thuật.

Khái niệm Stata học: outreg2, esttab, coefplot, putexcel, tabout

1. Cài đặt packages

* Lab 15 — Xuất bảng + biểu đồ
clear all
set more off
log using "lab15_<ten>.log", replace text

* Cài packages (chỉ chạy 1 lần)
ssc install outreg2, replace
ssc install estout, replace          // bao gồm esttab
ssc install coefplot, replace
ssc install tabout, replace

2. Hồi quy mẫu

* Dùng dataset auto có sẵn
sysuse auto, clear

* === 3 hồi quy ===
regress price mpg
estimates store m1

regress price mpg weight
estimates store m2

regress price mpg weight foreign
estimates store m3

3. Xuất bảng hồi quy bằng outreg2

* === outreg2: xuất ra Word/LaTeX ===
outreg2 [m1 m2 m3] using "regression_results.doc", replace ///
    title("Hồi quy giá xe theo các biến") ///
    addnote("*** p<0.01, ** p<0.05, * p<0.1") ///
    label

Output: file regression_results.doc mở được bằng Word, bảng chuẩn học thuật.

4. Xuất bảng bằng esttab (đẹp hơn)

* === esttab: chuẩn LaTeX/HTML ===
esttab m1 m2 m3 using "regression_esttab.rtf", replace ///
    title("Hồi quy giá xe — esttab style") ///
    cells(b(star fmt(3)) se(par fmt(3))) ///
    starlevels(* 0.10 ** 0.05 *** 0.01) ///
    stats(r2 N, labels("R-squared" "N")) ///
    label

Câu hỏi 1: So sánh outreg2 vs esttab — bạn thấy bảng nào dễ đọc hơn?

5. Coefficient plot (forest plot)

* === coefplot: biểu đồ hệ số ===
coefplot m1 m2 m3, ///
    drop(_cons) ///
    title("Coefficient Plot — 3 mô hình hồi quy") ///
    xtitle("Hệ số") ///
    xline(0, lcolor(red)) ///
    legend(rows(1)) ///
    graphregion(color(white))

graph export "coefplot.png", replace width(1400)

Câu hỏi 2: Coefficient plot có lợi gì so với bảng truyền thống?

6. Xuất bảng descriptive bằng tabout

* === tabout: xuất tabulate ra file ===
tabout foreign using "tab_foreign.html", replace ///
    cells(freq col) format(0 1) ///
    style(html) ///
    h1(Nguồn gốc xe)

* Hoặc summary stats
tabstat price mpg weight, statistics(mean sd min max) ///
    columns(statistics) ///
    save

7. Vẽ biểu đồ chuyên nghiệp

* === Theme và style đẹp ===
set scheme s2color
* Hoặc: set scheme white2 (cho slide thuyết trình)

* === Scatter với marker tùy chỉnh ===
twoway (scatter price mpg if foreign == 0, ///
        mcolor("114 47 55") msymbol(circle)) ///
       (scatter price mpg if foreign == 1, ///
        mcolor("0 88 163") msymbol(triangle)) ///
       (lfit price mpg, lcolor("46 125 80")), ///
       title("Giá xe theo MPG", size(medium)) ///
       subtitle("Phân biệt nội địa vs nhập khẩu", size(small)) ///
       xtitle("Mile per gallon") ///
       ytitle("Giá (USD)") ///
       legend(label(1 "Nội địa") label(2 "Nhập khẩu") label(3 "OLS fit") ///
              position(2) ring(0) cols(1)) ///
       graphregion(color(white)) ///
       plotregion(margin(small))

graph export "scatter_final.png", replace width(1400)

8. Xuất ra Excel bằng putexcel

* === Tạo file Excel có nhiều sheets ===
putexcel set "results.xlsx", replace
putexcel A1 = "Kết quả phân tích ", bold
putexcel A2 = "Lab 15 — Họ tên: <điền>"

* Sheet "Descriptive"
putexcel set "results.xlsx", sheet("Descriptive") modify
putexcel A1 = ("Biến", "Mean", "SD", "Min", "Max"), bold
putexcel A2 = "Price"
putexcel B2 = formula("=AVERAGE(...)")
* ... v.v

9. Mẹo trình bày kết quả Stata

# Mẹo
1 Bảng chỉ giữ kết quả quan trọng — không xuất tất cả output
2 Số liệu thập phân: 2-3 chữ số là đủ, đừng quá chi tiết
3 Sao biểu thị mức ý nghĩa: dùng quy ước rõ ràng
4 Notes dưới bảng: ghi rõ standard errors, mức ý nghĩa
5 Biểu đồ phải có: tiêu đề, nhãn trục, đơn vị, nguồn
6 Font + cỡ: đồng nhất trong toàn báo cáo
7 Màu sắc: tối đa 3-4 màu — quá nhiều gây rối

10. Liên hệ thuyết trình Buổi 15

Câu hỏi 3-5 (suy nghĩ + viết, 300 từ):

  • Bạn dùng outreg2 hay esttab cho slide thuyết trình? Tại sao?
  • Trong slide chỉ có 5-10 giây để đọc — bảng nên tối đa bao nhiêu hàng/cột?
  • Biểu đồ tốt cho slide khác biểu đồ tốt cho báo cáo — khác ở đâu?
  • Có nên dùng interactive plot (plotly)? Khi nào?
